# OXXYMORE INTERACTIVE

**Independent Game Studio & Interactive Systems Lab (France)**

OXXYMORE INTERACTIVE is a French independent studio dedicated to original strategic and educational game experiences.

## What we build
- Original IPs (board-inspired, digital-first, competitive-ready)
- Experimental game mechanics combining strategy, logic and impact
- Educational and cognition-oriented experiences
- Tools, prototypes and internal R&D

Flagship universe: **OBAN**.

## What this GitHub hosts
- Technical foundations and internal tooling
- Prototypes and experimental systems
- Shared libraries and reusable components
- Documentation repositories when relevant

Some repositories may remain private during active development phases.
